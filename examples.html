Example 1: Basic Image (Default Colors)

URL: /600x400

Generates a 600x400px image with default gray background and text showing dimensions.

Example 2: With Extension

URL: /600x400.png

Extensions are recognized but all images are returned as SVG.

Example 3: Path-based Colors

URL: /400x300/ff0000/ffffff

Red background (#ff0000) with white text (#ffffff).

Example 4: Custom Text via Query Parameter

URL: /500x350?text=Hello+World

Custom text using query parameter. Use + for spaces.

Example 5: Combined Path Colors and Query Text

URL: /600x400/0066cc/ffffff?text=Sample+Image

Blue background with white text and custom message.

Example 6: Query Parameters Override Path

URL: /400x300/000/fff?bg=ff6600&fg=000000&text=Override

Query parameters (orange bg, black fg) override path colors (black/white).

Example 7: 3-Character Hex Colors

URL: /250x150/f00/fff

Short-hand hex colors are automatically expanded (f00 ‚Üí ff0000, fff ‚Üí ffffff).

Example 8: Banner Image

URL: /800x200/333/fff?text=Banner+Image

Wide banner-style image with dark background.

Example 9: Square Avatar

URL: /200x200/9c27b0/ffffff?text=Avatar

Square image suitable for avatars or profile pictures.

Usage in HTML

<img src="YOUR_WORKER_URL/600x400/cccccc/969696" alt="Placeholder">
<img src="YOUR_WORKER_URL/600x400?text=Custom+Text" alt="Custom">

Usage in CSS

background-image: url('YOUR_WORKER_URL/1920x1080/000/fff?text=Hero+Image');
Compatibility: This service is compatible with dummyimage.com URL format. You can use query parameters (?text=, ?bg=, ?fg=) or path-based parameters interchangeably.
